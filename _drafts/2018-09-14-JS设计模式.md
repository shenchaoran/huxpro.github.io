# 适配器模式
M -> VM -> UI

# 桥接模式
举例：Array(100).fill(0).map(fn)
新建的Array根本不关心map会对其做了什么操作，具体有 fn 来决定。这样可以将实现部分和抽象部分分离开，是两者独立变化。
