# 概述
web component 的目的是
- 定制出自定义的 html 元素，扩展原生的标签元素，构建自己的 widgets
- 解决样式、脚本的作用域问题
- 提供组件的生命周期管理

组成：
- `Template`
- `Shadow DOM`
- `Custom Elements`
- `HTML Imports`
