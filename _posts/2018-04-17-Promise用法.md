# map

- 控制并发
- 返回的结果没有顺序

```
var Promise = require('bluebird');
const array = [];
for(let i=0;i<100;i++) {
    array.push(i);
}

Promise.map(array, (item) => {
    return Promise.resolve(item);
},{concurrency: 20})
    .then(rsts => {
        console.log(rsts);
    });
```
